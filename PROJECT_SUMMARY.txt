💰 EXPLICAÇÃO DE EV+
================================================================================

O que é EV+?
  Valor Esperado Plus = (Probabilidade do Modelo × Odds - 1) × 100

  EV+ Positivo = Lucro esperado a longo prazo
  EV+ Negativo = Perda esperada a longo prazo

Níveis de Sinal:
  ✓ FORTE COMPRA (EV+ > 5%)   → Expectativa positiva alta confiança
  → VALOR JUSTO (0-5%)        → Ligeiramente positivo ou equilíbrio
  ⚠ MARGINAL (-5% a 0%)       → Perto do equilíbrio, considere contexto
  ✗ EVITAR (< -5%)            → Expectativa negativa clara

Critério de Kelly:
  Percentual ótimo de aposta = (probabilidade × odds - 1) / (odds - 1)
  Recomendado: Use 25-50% de Kelly para reduzir variância

Exemplo:
  Previsão: 24,5 pontos
  Linha: 25,5 (precisa ir OVER)
  Odds: 1,95

  Percentual de Vitória do Modelo (Over 25,5): 48%
  EV+: -1,2% → Sinal MARGINAL
  Kelly: -0,012 → Não aposte ou use aposta pequena

🔧 ARQUITETURA TÉCNICA
================================================================================

Pipeline de Dados:
  CSV Load → Validação → Engenharia de Features → Normalização → Treinamento

Modelo:
  sklearn.linear_model.LinearRegression
  Features: 11 (normalizadas com StandardScaler)
  Pesos: Baseados em recência (recente > histórico)
  Alvo: Pontos Por Jogo (contínuo)

Previsão:
  Stats do Jogador → Vetor de Features → Escalado → Modelo → Previsão de Pontos
                                                                 ↓
                                                 Intervalo de Confiança (±σ)
                                                                 ↓
                                                 Análise de Tendências
                                                                 ↓
                                                 Cálculo de EV+

Cálculo de EV+:
  Previsão → Distribuição Normal (μ=pred, σ=2,39) → P(Over Linha)
                                                       ↓
                                         Comparar com Odds → EV%
                                                       ↓
                                         Critério de Kelly → Tamanho da Aposta
